* Device or platform (web, mobile, OS, browser if possible)
* User ID or username

   Store all activity logs in a separate database table called:
   activity_logs

Suggested fields:

* id
* user_id
* action_type (chat_message / file_upload / file_download)
* message_content
* file_name
* file_type
* file_size
* receiver_id
* ip_address
* device_info
* platform
* timestamp

3. Admin Dashboard

* View all chat logs
* View all file sharing logs
* Search logs by user
* Filter logs by date
* View IP addresses
* Download logs manually (CSV/PDF)

Step 1:
Collect all activity logs from the last 30 days.

Step 2:

* chat messages
* file sharing activity
* IP address
* device/platform
* timestamps
* users involved

Send the report automatically to this email address:
[manishsaini.p1@gmail.com](mailto:manishsaini.p1@gmail.com)

* Email is sent successfully
  AND
* Google Drive backup is completed

7. Scheduler
   Use a cron job or background worker to run this process automatically on the 1st day of every month.

8. Security

* Protect admin dashboard with authentication
